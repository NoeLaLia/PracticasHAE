# PracticasHAE
Prácticas de HAE (Hardware de Aplicación Específica) de tercero de carrera de Ingeniería Informática en la ESEI (Uvigo, Campus de Ourense)
Para el desarrollo del código utilicé mikroC PRO for PIC v7.5.0 y para la simulación Proteus 8 Profesional.
Por cada práctica se incluye el código .c en el que se desarrolla la práctica, el código .hex, código máquina en hexadecimal y archivo .pdsprj (Proteus Project) en el que se incluye la simulación con el código ya añadido.
Además, en cada práctica se incluye la descripción de la misma y material de apoyo para realizarla. 
El resto de ficheros .asm, .callertable etc es generado por el compilador mikroC y pueden ser desestimados.
